# My Portfolio Website

This is a modern portfolio website built with Next.js, React, TypeScript, and Tailwind CSS.

## Tech Stack

- **Framework**: Next.js 15 (App Router)
- **UI Library**: React 19
- **Language**: TypeScript
- **Styling**: Tailwind CSS v4
- **Animations**: Framer Motion
- **3D Graphics**: Three.js with React Three Fiber

## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Build

To create a production build:

```bash
npm run build
```

To start the production server:

```bash
npm start
```

## Features

- Modern, animated portfolio design
- Custom cursor with hover effects
- Smooth scroll animations with Framer Motion
- Responsive design
- Two main sections:
  - Homepage with Hero, About, Projects preview, and Contact sections
  - Projects page showing all projects
- Hash-based navigation support on homepage
- SEO-friendly with Next.js App Router
